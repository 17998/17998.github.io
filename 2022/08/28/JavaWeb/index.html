3.1.3 请求参数中文乱码

//POST请求在读取参数时是通过流，此处设置流的编码方法为UTF-8
req.setCharacterEncoding("utf-8");
String username = req.getParameter("username");
System.out.println(username);

//GET请求中，URL编码后与URL解码前的二进制数据是一样的
//1.先按照ISO_8859_1编码方式将乱码的字符串转为二进制字节数组
byte[] bytes = req.getParameter("username").getBytes(StandardCharsets.ISO_8859_1);
//2.再按照UTF-8的编码方式将二进制字节数组专为对字符串
String username = new String(bytes,StandardCharsets.UTF_8);
System.out.println(username);

resp.setContentType("text/html;charset=utf-8");

  • Axios是对原生的Ajax进行封装，简化书写

7.5.1 快速入门

//get请求
    axios({
        method:"get",
        url:"http://localhost:8080/Ajax/axiosServlet?username=zhangsan"
    }).then(function (response) {
        alert(response.data);
    })

//post请求
    axios({
        method:"post",
        url:"http://localhost:8080/Ajax/axiosServlet",
        data:"username=zhangsan"
    }).then(function (response) {
        alert(response.data);
    })

7.5.2 请求方式别名

//get请求
axios.get("http://localhost:8080/Ajax/axiosServlet?username=zhangsan").then(function (response) {
    alert(response.data);
})

//post请求
axios.post("http://localhost:8080/Ajax/axiosServlet","username=zhangsan").then(function (response){
    alert(response.data);
})
