  • 将本地仓库与远程仓库关联

  • git push –set-upstream origin master:master 将本地的master分支与远程的master分支进行绑定，下一次推送时可以直接使用git push

3.7 分支操作

  • git branch -d name 删除分支，在删除时做检查
  • git branch -D name 强制删除，在删除时不做检查
