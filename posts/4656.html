4.2调用函数

  1. 参数问题：JS可以传任意多个参数，也可以不传参数

  2. 假如不存在参数，如何规避？

    var abs = function(x){
     if(typeof x != 'Number')
         throw 'Not a Number';
     if(x>=0)
         return x;
     else
         return -x;
}

  3. 假如存在多个参数

    arguments

    arguments是JS免费赠送的关键字，代表传递进来的所有参数，是一个数组

    var abs = function (x) {
            var te = x;
            console.log("x="+x);
            for (let i=0;i<arguments.length;i++){
                console.log(arguments[i]);
            }
            if(x>=0)
                return x;
            else
                return -x;
        }

    问题：arguments包含所有的参数，如果有时候只想使用多余的参数，需要排除已有参数

    rest

    ES6引入的新特性，获取除了已经定义的参数外所有参数

    以前

    <script>
        function f(a,b) {
            console.log("a="+a);
            console.log("b="+b);
            if(arguments.length>2){
                for(let i=2;i<arguments.length;i++){
                    console.log(arguments[i]);
                }
            }
        }
    </script>

    新特性rest

    <script>
var f = function (a,b,...rest) {
            console.log("a="+a);
            console.log("b="+b);
            console.log(rest);
        }
    </script>

rest参数只能写在最后面，必须用...标识

this是无法指向的，默认指向调用它的那个对象

    var now = new Date().getFullYear();

}

    age: getAge
}

    name: 'xiaoming',

}
